使用红包注意以下几点：

  • 1.红包可以在test平台直抵交易费用，不能体现。
  • 2.每个红包限使用一次，不设找零。如您在使用没有完成支付，则该红包依然可以使用。
  • 3.每个订单只能只用一个红包，不能使用多个。
  • 4.每个红包都有一定的使用期限，请务必在使用期限内使用，以免逾期作废。
  • 5.如果在使用红包后进行退保操作，则退保后红包不予返还。
