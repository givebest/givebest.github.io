button

通用按钮组件，有蓝色、白色、约色、禁用。

蓝色按钮

@import "button.scss";

页面主操作 Normal 页面主操作 Disabled 蓝色中 蓝色小

白色按钮

@import "button.scss";

页面次要操作 Normal 页面次要操作 Disabled 白色中 白色小

红色按钮

@import "button.scss";

警告类操作 Normal 警告类操作 Disabled 红色中 红色小

蓝色边框按钮

@import "button.scss";

按钮 按钮 Disabled 边框蓝中 边框蓝小

黑色边框按钮

@import "button.scss";

按钮 按钮 Disabled 边框黑中 边框黑小

按钮组

@import "button.scss";

确定 取消
