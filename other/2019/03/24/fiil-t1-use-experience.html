Fiil T1 斐耳T1使用一周体验

缘起

首先 Fiil 的耳机已经买过两个，一个是头戴式的有线降噪耳机，用了段时间感觉还不错，后来感觉有线还是不方便，就慢慢搁置了。这个主打降噪，根据实际体验来说，在地铁上效果很不好，轰隆隆的嗓音还是很大，汽油公交上效果也不行，只有在电动公交上效果非常满意，开降噪听语音最小声音也能听清人声。还有就是平时中午睡觉时戴上效果也是满意的。

后来就是去年预订了 Fiil 随身星，挂在脖子上还是挺方便的，续航官方说是 11小时，平常通勤够用了。关于音质，看到网上有人说还不如 EarPods，这完全是不负责任的在黑，具体原因是坏了，联系客服得到答复是，修起来的价格差不多可以重新买一个了，这个质量可能是存在问题，我的是去年8月份买的，现在还没遇到问题（以后遇到问题再在这里更新吧）。收到后首先就是对比了 iPhone7 自带的 EarPods 耳机，听起来 Fiil 随身星解析更有层次、清晰，人声、乐器分的很清，开始感觉很惊艳，后来也就慢慢无感了。这个蓝牙断连在狭窄的空间很严重，比如：走廊、厕所间，接电话时用过几次，很不稳定，后来就没用它打过电话了。挂脖子上虽然感觉不到负重感，官方说是贴肤材料，还是感觉不方便。

预订

前段时间看到 Fiil T1 新品真无线耳机预售，就在 2019.3.6 付款预订了，在慢长等待中等到 3.16日商家发货。3.17收到货，看到是从深圳龙岗坂田发过来的，这跟官方客服口径不一致，客服说是会从浙江、广州两个菜鸟仓库发货。

收到后就迫不及待的打开试用了下，主要是和 Fiil 随身星对比，主要是因为木耳吧，选了多首音乐仔细重复对比了很多次，甚至是用 Fiil 随身星 听几秒，再切回 Fiil T1 回退几秒听同样细节。最终得出一个结论，T1 比随身星增强了空间感，大概就是听起来可以更好辨认乐器是从哪儿个方位发出来的（这个主要是对比随身星，纯属个人感官了）。

转眼今天已经3.24日了，Fiil T1 也使用了近一周了。因为首次预售的发货日期为3.15日，这过了一周首批用户都已收到货，各种使用报告、试用评测、评论已经很丰富，信息量大到足够各方面了解这款耳机了。所以这里就不评测了，就简单做个总结吧。

总结

使用了一周，总体感觉还不错。音质这东西比较主观，或者说是玄学，就放这里单独来讲。主要是对比 Fiil 随身星，下图可以看出 Fiil T1 比随身星出音口大了一圈，所以感觉听起来会更有空间感，就像网友说的听纯音乐很空灵的感觉。所以从随身星换成 T1 明显有种惊艳感。女生人声听起来很清晰，这是调音后的结果吧，但音量大点点听多了有时候甚至会感觉女声有些刺耳。下面分别介绍一下使用过程中非主观意识的优缺点。

优点

续航久

官方说单次使用6小时，加上充电盒可以充3次，可以满足24小时不间断使用。
实际使用，早晚通勤各1小时，上午挂耳朵上2小时左右，中午吃饭放充电盒里，中午睡觉到下班基本挂耳朵上(期间会有开会啥的就会放到充电盒里)，一天使用下来没遇到电量不足的情况。

连接快

官方说是开盒闪连，领先同行2秒。
实际使用时，只要手机开着蓝牙，打开充电盒（没有连接），取出耳机戴耳朵上，已经完成了连接。这速度真实场景使用没问题（比如：来电话了 => 打开充电盒 => 取出耳机 => 戴上 => 接通电话 => 不耽误通话）。

双耳 DSP 通话降噪

官方信息是，支持双耳高清通话，内置降噪专用麦克风，采用 DSP 技术消除环境噪音，在嘈杂的健身房也能享受高清通话。
实际使用，打电话时戴两个感觉不舒服(听诊器效应？)，我一般只用右耳，对方说听我说话很清楚，楼下还有小孩的玩闹声也能隐约听到，通话是没有问题的（还没在非常嘈杂的大街上试过）。现在打电话、语音就只用 Fiil T1 了，解放双手，也不耽误做些其它事。

佩戴牢固

这款耳机定位是运动耳机，4.8g重量，IP65 防水。最开始没注意到耳翼的作用，想要牢固非得使劲往耳朵里塞，太紧又不舒服，当时感觉是不是上当了？后来发现，戴上后，轻轻转一下耳机，用耳翼帮助固定，真的很牢固，猛甩头也不会掉的。这点官方介绍上没有看到详细说明，导致看到一些评论说是戴上就掉。感觉官方很有必要说明一下佩戴方法。

缺点

充电盒不显示电量，充电器有严格要求，Micro USB 充电接口

充电盒在不放耳机时充电时会有红灯亮起，充满电会灭灯。官方说明书里是这样介绍的：当充电盒红灯间隔闪烁3下，或打开充电盒时充电灯不亮时，需要给充电盒充电。还是无法确认充电盒实际的电量，只能保持常用常充的习惯了。
说明书上有个电池警告!说明，充电器必须是电压5V，电流<500mA 或者电脑 USB 口充电。开始没注意，用过 2A 和 1A 的充电器充过，还没遇到问题，建议还是首先电脑 USB 充电，毕竟 <500mA 的充电器在这个快充年代，很难找到。
还有就是充电口不是 Type-C 的。

听诊器

这个双耳要是戴的紧点，是非常明显的。我的解决方法就是，戴轻点缓解听诊器效果。

瞬间断连

就在办公室里听歌时，左耳即副耳机经常会突然没有声音然后很快又有了（不到一秒），在通勤路上没遇到，也有可能是断连太快周边不够安静没有留意到。
主耳机还是比较稳定的，有时候离开工位去倒水，因为隔了几道墙，提示蓝度断开，然后走回工位路上又会自动连接，然后继续播放。

副耳机不能单独使用，只能同时连接一个设备

基本上配对逻辑是这样的，成功连接一个设备，然后耳机就记下这个设备，再次开盖后搜索这个设备自动配对连接上了。要想连接其它设备，要按说明书重置设备(比较麻烦)，才能连接其它设备。

右耳即主耳机可以单独使用，即使把左耳机放到充电盒也可以。

解控复杂

通过左右两个耳机可以控制声音、上一首、下一首、呼唤 Siri、接挂电话、播放、暂停，因为是解控面板，只能通过繁琐操作比如三次左耳机是音量减，用过几次基本放弃，还是通过手机控制方便点。
