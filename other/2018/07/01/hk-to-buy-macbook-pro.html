Apple 香港官网购买 MacBook Pro 历程及使用体验

前言

用了 7 年多的华硕笔记本期间自己升级了硬件依然坚挺，也和当时买的顶配有关吧。除了显卡烧坏修过一次，部分键盘按键失灵，喇叭哑了之外竟然也没有别的毛病。不得不赞华硕的质量真不错。然而硬件太老，实际使用中总是慢的让人捉急，感慨硬件发展太快，顶配也很快会过时，不如买个一般配置用几年再换新的应该还划算些。

今年初终于下定决心准备换个新电脑，目标很明确就是 MacBook Pro。当然也犹豫过买 13 还是 15 ，查了很多资料也问了不少朋友、同事，最终决定买 13 的，主要原因是携带便捷性和价格，当然主要原因是穷。因为做开发想把内存升级到 16G。查了下各渠道的价格，包括某团、某先生、某闷等，定制 16G 内存比香港价格要贵1千左右而且没发票。考虑到维修等综合原因，就想直接从香港官方下单。查了下有转运公司可以从香港转运，也看到不少转运成功的晒单及教程。本来是想用这种方式，后面问到一个同事的朋友可以帮忙带过来，就放弃了转运方式。

等待

因2016、2017键盘易出问题有改进空间，网络上大部人推测 Apple WWDC 2018 有很大可能会更新 MacBook Pro，所以就多留意 WWDC 2018 相关信息等待 WWDC 2018 的到来。

很快，WWDC 2018 到来，但硬件并没有更新。近年苹果硬件、软件质量确有下降， iOS11 饱受诟病，于是今年重心是 iOS12，体验了下开发版确实快了很多。看来苹果是要从软件开始，一点点拉升用户体验，我想下一步就应该是升级改进硬件了，或许是今年秋，或者明年吧，不着急的可以观望一下。

下单

于是，在没等到新品的情况下，先是失望，然后就是迫不及待的下单了。网址：https://www.apple.com/hk/macbook-pro/，选好配置填好香港地址（同事朋友提供），用招行的双币信用卡付款。

订单已下达：2018.6.5

显示送货：1-3个工作天，送货时间：2018.6.8-2018.6.11

正在处理订单

记得很快，具体日期不记得了。

准备发货

这个应该是准备出厂了。

已发货：2018.6.7

顺丰发货，四川成都市发到￼香港。6.9 到香港并派件，当时是周末，地址、联络方式是留公司，所以改到下周一派送了。

收货 2018.6.12

因为周末原因，延迟了2天，同事朋友周一签收后，周二帮忙带过来，晚上到我手，算起来一周还是挺快的。拿到的时候保护膜还没撕开感觉很意外，看网上讲要打开电脑安几个 App 才好过关。当然帮忙归帮忙，喝茶费还是要有的 ：)。

总结

时间

下单：2018.6.5 => 发货：2018.6.7 => 派送：2018.6.9 ，因为是定制没有现货还是相当快的。

费用

用的招行 VISA 双币，这里汇率转换有个大坑：港币 => 美元 => 人民币，大约算了下汇率坑了￥200左右。

￥10859 + ￥500 = ￥11359

其它

苹果很激进，买的是 2017 13 低配版，只有两个雷电接口，又买了个绿联转接口 ¥158.00，3个 USB 和1个 HDMI，感觉够用了。

使用体验

总体感觉非常棒，轻薄，系统流畅，屏幕清晰，续航没有官方的长，键盘手感确实得熟悉一段时间。触摸板很好用，各种手势非常方便。安装了 PS 最新版很流畅，温度正常使用 40-50 吧，CPU 经常空置 80% 以上，内存长期 60-70 以上。

系统操作方式感觉没有 Windows 方便、便捷，学习成本较高，要配合很多快捷键、手势提高效率，Windows 就直观很多了。

工作软件够用，娱乐其它没有 Windows 丰富，一些 Windows 平台很好的软件没有 MacOS 版，但也基本能找到替换品，少了 Windows 的弹窗广告等各种打扰，更适合沉浸式工作。

总体来讲是个优秀的生产工具，性价比差，高配太贵了。

关于质量和耐用，我保持中立，上个华硕本也用了好多年，这样比较 MacBook 似乎优势不大。而且在使用其它苹果产品时也遇到过问题，比如 iPad Air2 耳机拔了仍是耳机模式没有外音，时好时坏的，因过保试了网上各种方法还是不行，干脆直接连接蓝牙音响用了。其它两个 iPhone 倒一直坚挺，没遇到啥问题。希望这个 MacBook 耐用吧。

写这篇文章时苹果已承认2016、2017款蝶式键盘问题，并承诺自激活起4年内出现问题免费更新：https://www.apple.com/cn/support/keyboard-service-program-for-macbook-and-macbook-pro/。不过根据苹果的尿性，保修可以，但其它地方不能坏，所以还是要爱惜一些为好。
